# 🤖 Crypto Monitor Bot - Binance & Discord

Um bot inteligente desenvolvido em **Python** que monitora o mercado de criptomoedas em tempo real através da API pública da **Binance** e dispara alertas automatizados de volatilidade diretamente em um canal do **Discord** via Webhooks.

---

## 🛠️ Tecnologias e Ferramentas

* **Linguagem:** Python 3.12
* **Bibliotecas:** `requests` (Consumo de APIs e envio de Webhooks), `time` (Gerenciamento de tempo de execução)
* **APIs Integradas:** Binance API (Market Data) & Discord Webhooks
* **Ambiente:** VS Code & Python Virtual Environments (`venv`)

Você precisará do Python instalado e de um link de Webhook de um canal do Discord.
